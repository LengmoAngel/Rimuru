# wwdg_basic

### 实验流程

- 初始化 BOARD_WWDG_PORT ，配置预分频系数（Prescaler），重装载计数器上限窗口值（UpperLimit），使能WWDG_INT_ALMOST_TIMEOUT中断。
- 中断处理函数中判断是否已经重装载计数器到指定次数，否则重装载一次WWDG计数器，并记录重装载次数，是则不再重装载计数器。
- 主函数中不作任何处理，在初始化后进入死循环。经过一段延时后 ，串口将打印一个 “.” ，表示CPU仍在工作。
